Projects

Raptor Tracker

Full stack interactive web application for exploring locations of GPS locations of golden eagles and African vultures tracked with solar powered GPS trackers by HawkWatch International. This JavaScript app was built using React, LeafletJS, NodeJS, Express, PostgreSQL and CSS.

Live Page Demo View GitHub Repository

Process Journal

Full stack interactive web application for setting goals and tracking progress. This application is based on the premise that in order to acheive long term goals, you should focus on maintaining good habits related to your goal as well as tracking measurable steps toward your goal. This JavaScript app was built using React, NodeJS, Express, PostgreSQL, ChartJS and CSS.

Live Page Demo View GitHub Repository

Interactive Quiz App

This is an interactive quiz application featuring information about forest and tree ecology including ecosystem services and human interactions. This quiz app was built using jQuery, plain JavaScript, HTML and CSS.

Live Page Demo View GitHub Repository

Global Earthquake Explorer

This web application allows users to spatially explore recent earthquakes from the USGS global earthquake API. The map was built with Leaflet.js, jQuery and uses USGS global earthquake data as well as a global faults layer.
