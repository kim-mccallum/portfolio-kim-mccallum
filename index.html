About Me

Projects

Raptor Tracker

A full-stack interactive web application for exploring locations of GPS locations of golden eagles and African vultures tracked with solar-powered GPS trackers by HawkWatch International. I built this application as a volunteer for HawkWatch International in order to provide the public and researchers a means of visualizing raptor habitat use and behavior. This is a JavaScript app built on React, LeafletJS, NodeJS, Express, PostgreSQL, and CSS.

Process Journal

A full-stack interactive web application for setting goals and tracking progress. This application is based on the premise that in order to achieve long-term goals, you should focus on maintaining good habits related to the goal and also tracking measurable steps toward it. I built this app for myself and friends as a tool to improve focus and balance around goal-setting and tracking. This is a JavaScript app built on React, NodeJS, Express, PostgreSQL, ChartJS and CSS.

Global Earthquake Explorer

This is a front-end web application that allows users to spatially explore recent global earthquake data in the context of global fault locations. I built this after learning about the phenomenon of induced seismicity from a researcher friend of mine. It is intended as a visualization tool for the general public to observe the distribution of earthquakes relative to faults as well as human development. The map was built with Leaflet.js, jQuery, and uses USGS global earthquake data as well as a global fault layer.
